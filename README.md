# 📝 Notes API (FastAPI + PostgreSQL + Docker + CI/CD)

A production-ready Notes API built with **FastAPI**, **PostgreSQL**, and fully containerized using **Docker**.  
Includes **automated testing, CI/CD with GitHub Actions, and Docker image deployment to Docker Hub**.

---

## 🚀 Features

- 📌 Create, read, update, and delete notes
- ⚡ FastAPI for high-performance APIs
- 🐘 PostgreSQL database
- 🐳 Dockerized application
- 🔄 CI pipeline (lint + tests with pytest)
- 🚀 CD pipeline (auto build & push Docker image)
- 🧪 Automated testing with pytest
- 📦 Clean project structure

---

## 🏗️ Tech Stack

- **Backend**: FastAPI
- **Database**: PostgreSQL
- **ORM**: SQLModel
- **Containerization**: Docker & Docker Compose
- **CI/CD**: GitHub Actions
- **Testing**: Pytest
